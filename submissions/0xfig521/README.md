## Description / 项目简介

TradeTrace is a trade transparency and proof platform on Sui. It uses DeepBook for execution verification and Walrus for immutable evidence storage, providing a complete audit trail for trades with on-chain verification and off-chain evidence anchoring.

TradeTrace 是一个基于 Sui 的交易透明与证明平台。利用 DeepBook 进行执行验证，Walrus 提供不可篡改的证据存储，为交易提供完整的审计链。
