## Description / 项目简介

SuiTrustPay is a payment and wallet solution on Sui with a Walrus-powered evidence layer for trust and transparency. It enables secure peer-to-peer payments with verifiable transaction receipts stored immutably on Walrus.

SuiTrustPay 是基于 Sui 的支付与钱包解决方案，通过 Walrus 证据层实现信任与透明。支持点对点安全支付，交易回执不可篡改地存储在 Walrus 上。
