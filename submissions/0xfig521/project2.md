## Description / 项目简介

DeepBook Intent Router enables intelligent order routing across DeepBook pools on Sui. It analyzes liquidity depth, price impact, and gas costs to find the optimal execution path for each trade intent.

DeepBook Intent Router 实现基于 Sui DeepBook 的智能订单路由。分析流动性深度、价格影响和 Gas 成本，为每笔交易找到最优执行路径。
