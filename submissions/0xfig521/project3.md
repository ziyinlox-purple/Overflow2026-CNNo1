## Description / 项目简介

Sui Actions / PTB Links enables shareable programmable transaction blocks for Sui onboarding and developer experience. Inspired by Solana Blinks, it allows users to create and share PTB-based action links.

Sui Actions / PTB Links 实现可分享的可编程交易块，优化 Sui 的开发者体验。受 Solana Blinks 启发，用户可创建并分享基于 PTB 的操作链接。
