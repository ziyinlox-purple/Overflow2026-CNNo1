## Description / 项目简介

<!-- 一段话描述你的项目：做什么，为什么重要 / What it does, why it matters -->

NARWHAL（独角鲸表单）是一个原生于 Walrus、Seal、Sui 的去中心化反馈与表单平台——任何人都能在 30 秒内造出一张「美观漂亮、且永远归你所有」的表单，开箱即用 10 种字段类型覆盖几乎所有场景（Bug 报告、功能请求、用户调研、申请审核等）；三件套各司其职：Walrus 存储、Seal 加密、Sui 确权，你既可以整表加密、也可以只加密敏感字段，其他人永远只看到密文。它的价值在于：传统 SaaS 表单（Typeform、Google Forms）把你的数据锁在别人的服务器里、随时可能被读取或停服，而 NARWHAL 做到无中心服务器、无后端账户、无供应商锁定——每一次创建、提交、批改都在链上留有可审计回执，权限是一个能转让、能核销的 AdminCap 对象而非数据库里的一行 is_admin，让用户敢于填写真实而敏感的反馈（因为只有创建者能解密），也让组织真正「听见冰层之下的声音」。
