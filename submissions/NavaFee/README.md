## Description / 项目简介

**Call BTC up or down, get paid automatically — CallIt turns DeepBook Predict into a game anyone can play in ten seconds.**

Call BTC **UP** or **DOWN**, stake a few dUSDC, and the oracle settles you automatically. Every odd, every payout, and every settlement on screen is on-chain DeepBook Predict truth — wrapped in a one-thumb arcade loop with win streaks, badges, a weekly leaderboard, and Telegram result cards that pull the next player in. Two form factors, one account: a web PWA and a Telegram Mini App, both using Telegram as a one-tap login (no phone number, no password, no seed phrase) over a sponsored session wallet — ten seconds from landing to a funded on-chain account. Live odds shimmer as the vol surface updates; if an oracle feed goes stale, betting honestly disables itself instead of failing a transaction.

> 把 DeepBook Predict 做成人人十秒上手的游戏:看涨还是看跌 BTC,下注几个 dUSDC,预言机到期自动结算。屏幕上每一个赔率、每一笔赔付都是链上协议的真实数据。
