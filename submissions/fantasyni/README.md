## Description / 项目简介

SuiAgentWallet is an AI Agent Wallet powered by Sui blockchain, featuring zkLogin passwordless auth, Deepbook Dex trading, and on-chain delegation-based fund management

Sui Core Features:

🚀: ZkLogin Agent Wallet
Google login, zero private key management

🦄: PTB Execution
Backend tools return PTB -> frontend buildTransactionFromPtb builds -> zkLogin signing -> on-chain submission

💼: Delegation List
Filtered by current wallet address, live on-chain status refresh
