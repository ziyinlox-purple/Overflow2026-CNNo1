## Description / 项目简介

Suix is a desktop AI coding agent purpose-built for Sui Move development. It combines an LLM-powered conversational interface with 24+ built-in tools and 15 embedded Sui Move skill modules, enabling developers to create, compile, test, deploy, and debug Move smart contracts through natural language interaction.
